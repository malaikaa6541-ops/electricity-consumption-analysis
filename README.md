# ⚡ Electricity Consumption Analysis – Islamabad

## 📌 Project Overview
This project focuses on analyzing household electricity consumption patterns in Islamabad from 2023 onwards. The goal is to understand how energy use varies with environmental and weather conditions.

## 🧠 Objective
- Analyze household electricity consumption trends
- Identify seasonal and weather-based consumption patterns
- Explore the correlation between weather conditions and energy usage

## 🛠️ Tools & Technologies
- Python 🐍
- Pandas & NumPy
- Matplotlib & Seaborn
- Jupyter Notebook
- Weather datasets (WeatherWalay collaboration)

## 📊 Dataset
- Household electricity consumption data (Islamabad)
- Weather data (temperature, humidity, etc.) from WeatherWalay
- Time period: 2023 – Present

## 🔍 Key Analysis
- Daily and monthly consumption trends
- Peak usage time identification
- Impact of temperature and weather on electricity demand
- Correlation analysis between weather variables and energy usage

## 📈 Results & Insights
- Higher electricity usage observed during extreme temperatures
- Strong correlation between temperature and energy consumption
- Seasonal variations significantly affect household demand
